INSTALACIÓN DE DOTFILES (Ubuntu / GNOME)
=========================================

REQUISITOS
----------
- Ubuntu 22.04 o superior
- Conexión a internet
- Git instalado (sudo apt install git)

PASOS
-----

1. Clonar el repositorio:
   git clone <tu-repo> ~/dotfiles
   cd ~/dotfiles

2. Instalar paquetes esenciales:
   chmod +x scripts/*
   ./scripts/install-packages

3. Instalar dotfiles (crea symlinks):
   ./scripts/install-dotfiles

4. Configurar entorno de desarrollo:
   ./scripts/setup-dev-env

5. Cambiar shell a zsh:
   chsh -s $(which zsh)

6. Cerrar sesión y volver a entrar

SCRIPTS DISPONIBLES
===================

- install-packages  Instala todos los paquetes necesarios (apt, snap, binarios)
- install-dotfiles  Crea symlinks de configuraciones en $HOME
- setup-dev-env     Configura entornos de desarrollo (Node, Go, Rust, etc.)
- backup-dotfiles   Crea backup de configuraciones actuales
- sync-dotfiles     Sincroniza cambios con Git
- update-all        Actualiza sistema y herramientas

PERSONALIZACIÓN
===============

1. Edita .gitconfig con tu nombre y email
2. Ajusta .zprofile para tus variables de entorno
3. Modifica aliases en .zshrc según tus necesidades
4. Personaliza Starship en .config/starship.toml

NOTAS UBUNTU/GNOME
==================

- En Ubuntu, 'bat' se instala como 'batcat' y 'fd' como 'fdfind'.
  El script install-packages crea los symlinks necesarios en ~/.local/bin.
- Kitty está configurado con linux_display_server auto para funcionar
  tanto en sesiones X11 como Wayland de GNOME.
- Los configs de i3 y Hyprland están incluidos pero no son necesarios
  si usas GNOME como entorno de escritorio.
- gnome-tweaks se instala para personalización adicional de GNOME.

TEMA
====

Todos los archivos usan el tema Catppuccin Mocha para consistencia visual.
