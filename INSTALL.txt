INSTALACIÓN DE DOTFILES (Ubuntu 24.04+ / 26.04)
=================================================

REQUISITOS
----------
- Ubuntu 24.04 o superior
- Conexión a internet
- Git: sudo apt install git

INSTALACIÓN
-----------
git clone <tu-repo> ~/dotfiles
cd ~/dotfiles
./install.sh

El script es interactivo: te pregunta qué instalar y explica cada componente.

DESPUÉS DE INSTALAR
-------------------
1. Cierra sesión y vuelve a entrar (para que Zsh tome efecto)
2. Edita ~/.gitconfig con tu nombre y email
3. Ejecuta: tldr --update (si instalaste tealdeer)

SCRIPTS
-------
- install.sh                  Instalador interactivo principal
- scripts/update-all          Actualiza sistema completo (alias: upd)
- scripts/check-dependencies  Verifica qué herramientas están instaladas

SINCRONIZAR CAMBIOS
-------------------
Usa lazygit (lg) desde ~/dotfiles para commitear y pushear.
